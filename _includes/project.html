Project

PhotoPlace is a desktop program to easily geotag and create a Google Earth layers with your photos. Below you can see the main features of this project:

Geotag your Photos

By providing a GPX document from a GPS device you can geotag all the photos within a folder. Even, by using a CSV plugin it is also posible to import the geodata for the images.

Create Google Earth Tour

It can generate a KMZ layer for Google Earth, with all photos included. Moreover, you can create a presentation with audio, a virtual tour flying to each image.

Configurable and Extensible

The information displayed is highly configurable with HTML templates and its functionality can be complemented with plugins.

OpenSource and Multiplatform

With binaries for Windows and packages for Debian/Ubuntu. Made with Python 2.x you can download and run it directly in the folder.
