Downloads

PhotoPlace binaries are available at the GitHub releases section. Packages with the source code and all the contents of the repo are also available for downloading there.

The latest version is and it was updated . Here you can download directly the packages:

  • Installer for Windows platforms (Vista, 7, 8).
  • GNU/Linux package for Debian/Ubuntu.
  • Compressed zip binary for Windows platforms (no installer).
  • Compressed tar.gz package for other Linux/UNIX/OSX platforms.

On GitHub, across time, all PhotoPlace versions were downloaded times.


After download the right package for your platform on the previous step, the installation is pretty straight forward; for Windows, just accept the license and click next and next ... -keep calm, there is no malware in the installer-. For Ubuntu/Debian, just click on the file and the official installer should start, otherwise, to install it manually, open a terminal and within the download folder type: sudo dpkg -i photoplace*.deb; sudo apt-get install -f.
